Création d'un compte GitHub
Comment se créer un compte ?

Avant de commencer à utiliser Git ou même GitHub, il faut commencer par ce créer un compte. GitHub propose 4 différentes solutions qui varient par la protection et la confidentialité des projets :

  • Offre gratuite : les projets sont visibles et téléchargeables par tous les utilisateurs.
  • Offre personnel : possibilité de crééer des projets privés visible seulement par les membres invités.
  • Offre organisation : possibilité de gérer les utilisateurs avec des niveaux de permissions.
  • Offre entreprise : possibilité d’héberger les projets sur ses propres serveurs avec des protocoles de sécurité propre à l’entreprise.

Vu que nous débutons, nous allons commencer avec un compte simple. Rendez vous donc sur la page d'acceuil :


Il ne vous reste plus qu’à vous trouver un identifiant (1), renseigner votre adresse mail (2), vous trouver un mot de passe (3) et on valide (4) !

Vous accèderez à une page vous permettant de choisir le type de votre compte (les offres présentées précedemment). Ici nous n’avons rien à faire, il suffit de valider et c’est terminé. Vous avez enfin votre compte GitHub. Vous allez pouvoir créer vos propres projets ou alors rejoindre les projets des membres de la communauté.

Linux

Il faut avant tout installer plusieurs paquets sur votre ordinateur si cela n’est pas déjà fait :

  • Pour Linux Fedora, il faut utiliser “yum” et donc lancer cette commande dans un terminal :
  • Pour les autres Linux il faut utiliser le apt-get comme suit toujours dans un terminal :

Pour pouvoir utiliser Git sur Linux, il suffit d’installer le paquet Git. On peut l’installer de plusieurs façons :

  • Depuis un navigateur web, il suffit de se rendre sur le site puis de lancer la bonne commande dans un terminal selon votre version de Linux.
  • Depuis le gestionnaire des paquets, il suffit de rechercher “git” et de lancer l’installation.
  • Depuis Logiciels qui est un logiciel de remplacement de logithèque pour ubuntu 16.04 Xenial. Même démarche qu’au dessus.
  • Depuis un navigateur web, trouvez un client du genre "apt://git" cliquer dessus permettra d’ouvrir une fenêtre qui vous demandera si vous voulez installer Git.



Windows

L’installation de Git sur windows est très simple. Il suffit de télécharger et de lancer l’éxécutable à cette adresse

Vous allez ainsi installer deux programmes :

  • Git GUI : une interface graphique pour Git offrant toutes les fonctionnalités de la version console ainsi que des fonctionnalités visuelles
  • Git BASH : il s’agit de la version ligne de commande. Ce shell doit être utilisé car il permet d’utiliser toutes les commandes de Git.
Cette installation s’intègre au shell windows et permet de lancer Git depuis n’importe quel dossier avec un simple clic droit.



MacOS

La première chose à faire pour utiliser Git est de l’installer sur votre ordinateur. Pour celà il y a deux manières de faire :

  • Installer à partir des sources
  • Installer à partir d’un paquet existant sur votre ordinateur
Nous vous conseillons d’installer Git depuis les sources, en vous dirigeant sur la page suivante.

Cliquez sur “Download for Mac”, le téléchargement démarrera ensuite (si ce n’est pas le cas suivez les instructions de la page).

Vous avez alors télécharger un fichier disque .dmg que vous pourrez exécuter une fois le téléchargement terminer. Vous pourrez alors procéder à l’installation de Git sur votre Mac en cliquant sur le fichier .pkg (fichier package qui sera installé sur votre ordinateur). Une fois le package installé, vous pourrez utiliser Git sur votre ordinateur.
