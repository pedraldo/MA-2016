Introduction

Un logiciel de gestion de version de code est un outil incontournable pour tout developpeur afin de ne pas perdre des jours de travail. Ce type de logiciel permet de garder toutes les traces de modifications apportées sur le projet. Il en existe nombreux. Certains sont payant d'autres gratuits.

Git est un logiciel de gestion de version de code totalement gratuit et facile à comprendre. Dans ce manuel nous allons vous montrer comment utiliser git afin de faciliter la gestion des versions de votre code.

Ce manuel comprend 3 grandes parties:

  • Partie 1: Installation et configuration de git
  • Partie 2: Gestion de projet avec git
  • Partie 3: Quelques astuces sur l'utilisation de git
