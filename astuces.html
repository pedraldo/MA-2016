Auto-Complétion

Si vous utilisez Windows vous l’avez déjà installé avec Bash Git. Si vous utilisez MacOS ou Linux vous en aurez peut être besoin cela facilite la vie.

Pour l’avoir c’est simple, vous devez télécharger le code source à cette adresse.

POUR UN UTILISATEUR : Vous devez ensuite l'ajouter dans votre répertoire personnel en le renommant: ".git-completion.bash" et ajouter cette ligne à votre fichier ".bashrc" : "source ~/.git-completion.bash" Ne pas prendre les guillemets en compte

POUR TOUS LES UTILISATEURS : Vous devez copier le fichier de git en le nommant : "git-completion.bash" et l'ajouter au répertoire /etc/bash_completion.d pour Linux et au répertoire /opt/local/etc/bash_completion.d pour MacOS

Pour utiliser l'auto complétion c'est simple il suffit d'appuyer sur la touche “tab” lorsque que vous commencer un mot.
